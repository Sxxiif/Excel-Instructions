# Contributing

Contributions are welcome.

## How to Contribute

1. Fork this repository.
2. Create a branch:

```bash
git checkout -b improve-formulas
```

3. Make your changes.
4. Commit:

```bash
git commit -m "Improve formula examples"
```

5. Push:

```bash
git push origin improve-formulas
```

6. Open a pull request.

## Contribution Rules

- Use fictional data only.
- Never upload confidential company information.
- Test formulas in Excel before submitting.
- Explain each formula in simple English.
- Mention when a function requires Microsoft 365 or a newer Excel release.
