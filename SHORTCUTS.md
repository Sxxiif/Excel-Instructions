# Essential Excel Keyboard Shortcuts

## General

| Shortcut | Action |
|---|---|
| `Ctrl + N` | New workbook |
| `Ctrl + O` | Open workbook |
| `Ctrl + S` | Save |
| `F12` | Save As |
| `Ctrl + P` | Print |
| `Ctrl + Z` | Undo |
| `Ctrl + Y` | Redo |
| `Ctrl + C` | Copy |
| `Ctrl + X` | Cut |
| `Ctrl + V` | Paste |
| `Ctrl + F` | Find |
| `Ctrl + H` | Replace |

## Data and Tables

| Shortcut | Action |
|---|---|
| `Ctrl + T` | Create an Excel Table |
| `Ctrl + Shift + L` | Turn filters on/off |
| `Alt + Down Arrow` | Open filter or dropdown |
| `Ctrl + Space` | Select column |
| `Shift + Space` | Select row |
| `Ctrl + A` | Select current data region |
| `Ctrl + Arrow` | Jump to edge of data |
| `Ctrl + Shift + Arrow` | Select to edge of data |

## Formulas

| Shortcut | Action |
|---|---|
| `F2` | Edit active cell |
| `Alt + =` | AutoSum |
| `Ctrl + `` | Show/hide formulas |
| `F4` | Cycle relative/absolute references |
| `Ctrl + Enter` | Fill selected cells |
| `Ctrl + D` | Fill down |
| `Ctrl + R` | Fill right |

## Formatting

| Shortcut | Action |
|---|---|
| `Ctrl + 1` | Format Cells |
| `Ctrl + B` | Bold |
| `Ctrl + I` | Italic |
| `Ctrl + U` | Underline |
| `Ctrl + 5` | Strikethrough |
| `Ctrl + Shift + %` | Percentage |
| `Ctrl + Shift + #` | Date format |
| `Alt + H, O, I` | AutoFit column width |
| `Alt + H, O, A` | AutoFit row height |

## Sheets

| Shortcut | Action |
|---|---|
| `Shift + F11` | New worksheet |
| `Ctrl + Page Up` | Previous sheet |
| `Ctrl + Page Down` | Next sheet |
